Este repositório contém a tradução não oficial para o português brasileiro do cliente do Hytale.

Como instalar

1. Clone este repositório
2. Copie o arquivo client.lang e meta.lang para a pasta "C:\Users\{seu usuario}\AppData\Roaming\Hytale\install\release\package\game\latest\Client\Data\Shared\Language\pt-BR\" (Crie a pasta pt-BR se necessário)
3. Reinicie o Hytale
4. Selecione a língua "Portugues (PT-BR)" no menu de seleção de línguas

Caso queira desinstalar, basta apagar a pasta pt-BR.
